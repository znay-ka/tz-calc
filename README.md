# Демо проект "Калькулятор займа".

Текст задания в файле Тестовое задание PHP.doc

###### Подъём проекта:
```
git clone https://gitlab.com/clown.geo/calc.git calc 
cd calc 
php composer.phar install 
cd public 
php -S localhost:8888 -t ./ index.php
```
